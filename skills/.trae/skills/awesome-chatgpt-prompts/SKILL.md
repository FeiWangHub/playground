# Awesome ChatGPT Prompts

This skill provides access to a massive collection of prompts from the [Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts) repository.

## Capabilities

- Search for prompts by keyword (e.g., "linux terminal", "translator", "interviewer").
- Get detailed prompt templates to act as specific personas.

## Usage

Search for a prompt:

```bash
python3 .shared/awesome-chatgpt-prompts/scripts/search.py "<keyword>"
```
