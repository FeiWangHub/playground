# skill-dev-skill

Scaffold new skills for the playground.

## Prerequisites

Python 3 installed.

## How to Use

Run the creation script with the skill name and description:

```bash
python3 .shared/skill-dev-skill/scripts/create_skill.py "skill-name" "Skill description"
```

Example:

```bash
python3 .shared/skill-dev-skill/scripts/create_skill.py "weather-skill" "Get weather updates"
```
