# awesome-chatgpt-prompts

Search and retrieve prompts from the Awesome ChatGPT Prompts collection.

## Prerequisites

Check if Python is installed:

```bash
python3 --version || python --version
```

## How to Use

When you need a specific persona or prompt template for ChatGPT/Claude, search the database:

```bash
python3 .shared/awesome-chatgpt-prompts/scripts/search.py "<keyword>"
```

Example:

```bash
python3 .shared/awesome-chatgpt-prompts/scripts/search.py "linux"
python3 .shared/awesome-chatgpt-prompts/scripts/search.py "javascript"
```
