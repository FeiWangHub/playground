# pptx-skill

Generate PowerPoint presentations from markdown content.

## Prerequisites

Install the `python-pptx` library:

```bash
pip install python-pptx
```

## How to Use

1. Create a markdown file with your slide content (headers are slide titles).
2. Run the generation script:

```bash
python3 .shared/pptx-skill/scripts/generate_pptx.py "input.md" "output.pptx"
```
